## What is Jumla-bot?

**Jumla-bot** is an AI-powered wholesaling automation ecosystem designed to help real estate wholesalers operate faster, smarter, and at scale.

Instead of relying on manual follow-ups, spreadsheets, and constant back-and-forth, Jumla-bot automates the **front-end and operational side** of wholesaling while keeping humans in control of negotiations and legal decisions.

This is **not just a chatbot**.

Jumla-bot is a **virtual wholesaling assistant + CRM + automation engine**.

## Architecture (High Level)

* **Frontend:** React
* **Backend:** FastAPI (Python)
* **Database:** PostgreSQL
* **AI Layer:** LLM APIs (agent-based)
* **Messaging:** SMS + Email integrations
* **Integrations:** Property data & calendar APIs

Built for scalability and team-based development.
